# T-Guardian: Vision 모델 학습 데이터셋 확보 전략 보고서

본 보고서는 "합성(Synthetic)" 방식 외에, 인터넷상에 공개된 **실제 로고 탐지 및 위조품 데이터셋(Open Datasets)**을 조사하여 T-Guardian의 Vision 모델 학습에 즉시 활용 가능한 리소스를 정리한 문서입니다.

---

## 1. 핵심 결론 (Executive Summary)

인터넷상에는 상표 침해 및 로고 탐지를 위한 훌륭한 오픈소스 데이터셋들이 이미 존재합니다. "처음부터 100% 합성"할 필요 없이, 아래의 데이터셋을 **초기 학습(Pre-training)**용으로 사용하여 모델의 '눈'을 먼저 뜨게 하고, 그 후 합성 데이터로 **미세 조정(Fine-tuning)**하는 하이브리드 전략이 가장 효율적입니다.

### 🔍 발견된 주요 데이터셋 (Best Picks)
1.  **LogoDet-3K:** 가장 방대한 로고 탐지 데이터셋 (15만 장, 3000개 브랜드). 모델의 기초 체력을 기르는 데 최적.
2.  **TMID (Trademark Infringement Detection):** 알리페이(Alipay)에서 수집된 실제 침해 사례 1.7만 쌍. '유사 상표' 판단 로직 학습에 필수.
3.  **Fake/Real Logo Detection Dataset:** 진짜 vs 가짜 로고 분류를 위해 정제된 데이터셋.

---

## 2. 활용 가능한 공개 데이터셋 상세 분석

### 2.1 LogoDet-3K (가장 강력한 기초 자료)
*   **출처:** [DatasetNinja / GitHub](https://datasetninja.com/logodet-3k)
*   **규모:** 이미지 158,652장 / 3,000개 로고 카테고리 / 194,261개 바운딩 박스(Annotation)
*   **특징:**
    *   스포츠, 식품, 의류 등 9개 카테고리로 분류됨.
    *   로고가 작거나 흐릿한 **'어려운 사례(Hard Examples)'**가 많이 포함되어 있어 실전 훈련에 좋음.
*   **활용법:**
    *   T-Guardian의 객체 탐지(Object Detection) 모델(YOLOv8 등)을 이 데이터로 먼저 학습시키면, **"이미지에서 로고가 어디에 있는지"** 찾는 능력을 갖추게 됩니다. (Pre-training Task)

### 2.2 TMID (유사 침해 판단 특화)
*   **출처:** [EMNLP 2023 논문 / GitHub](https://github.com/emnlpTMID/emnlpTMID.github.io)
*   **규모:** 17,365 쌍(Pair)의 상표 데이터 (등록 상표 vs 침해 의심 상표)
*   **특징:**
    *   단순 로고 탐지가 아니라, **"이 상표가 저 상표를 침해했는가?"**라는 법적 판단(Legal Reasoning)을 위해 구축된 데이터셋.
    *   **'유사한데 침해인 것'**과 **'유사한데 침해가 아닌 것'**이 라벨링되어 있어 Metric Learning에 최적.
*   **활용법:**
    *   Vision 모델의 **Metric Learning (유사도 거리 학습)** 단계에서 핵심 데이터로 사용.
    *   "스타벅스"와 "스타박스"가 왜 비슷한지 기계가 이해하도록 가르치는 교과서.

### 2.3 Fake/Real Logo Detection Dataset (진품/가품 분류)
*   **출처:** [Kaggle](https://www.kaggle.com/datasets/prosperchuks/fakereal-logo-detection-dataset)
*   **규모:** 약 825개 브랜드 로고 (Resize된 70x70 이미지)
*   **특징:**
    *   진짜(Genuine) 로고와 가짜(Fake) 로고가 짝지어 있음. (약 67%가 가짜 데이터)
*   **활용법:**
    *   데이터 양은 적지만, **'미세한 변형(Distortion)'**을 감지하는 모델 튜닝용으로 사용.

### 2.4 기타 유용한 리소스
*   **Open Brand Logos (QMUL-OpenLogo):** 27,000장의 로고 이미지. 다양한 각도와 조명 환경 포함.
*   **Innovatiana Luxury Fashion Fraud Detection:** 샤넬(Chanel) 등 명품 브랜드의 진품/가품 이미지. 패션 카테고리 특화 모델 학습 시 유용.

## 3. 수정된 Vision 모델 학습 로드맵 (Data Pipeline v2.0)

기존의 "100% 합성" 전략을 수정하여, **[공개 데이터셋 → 합성 데이터 → 실 데이터]**의 3단계 파이프라인을 제안합니다.

### Phase 1: 기초 근육 만들기 (Pre-training)
*   **데이터:** **LogoDet-3K (15만 장)**
*   **목표:** "세상의 모든 로고를 찾아내라."
*   **학습:** YOLOv8 또는 EfficientDet 모델 학습.
*   **결과:** 간판, 티셔츠, 컵 사진을 주면 브랜드 상관없이 로고 위치(Bounding Box)를 정확히 찾아냄.

### Phase 2: 비교 능력 기르기 (Metric Learning)
*   **데이터:** **TMID (1.7만 쌍) + Fake/Real Dataset**
*   **목표:** "이거랑 저거랑 얼마나 비슷해?"
*   **학습:** Siamese Network (Triplet Loss).
*   **결과:** 브랜드 A와 브랜드 B의 시각적 유사도를 0~100% 숫자로 산출.

### Phase 3: 타겟 브랜드 적응 (Domain Adaptation) - **여기가 합성 필요**
*   **데이터:** **합성 데이터 (Target Synthetic)**
*   **목표:** "스타벅스 짝퉁을 기가 막히게 찾아내라."
*   **행동:** 우리가 감시할 고객사(예: 스타벅스, 삼성)의 로고만 가지고 합성 데이터 1만 장 생성.
*   **학습:** Phase 2 모델을 이 데이터로 **Fine-tuning**.
*   **이유:** 공개 데이터셋에는 내 고객사 로고가 없을 수도 있고, 있어도 충분치 않기 때문.

## 4. 결론

사용자님의 직관이 맞았습니다. **우리가 처음부터 맨땅에 헤딩할 필요가 없습니다.**
이미 학계와 오픈소스 커뮤니티에 **'로고 탐지'**와 **'침해 판단'**을 위한 훌륭한 데이터셋이 존재합니다.

**전략 수정:**
1.  **LogoDet-3K**를 다운로드 받아 Vision 모델의 '기본기'를 다진다.
2.  **TMID** 데이터로 '유사도 비교' 로직을 학습시킨다.
3.  **합성 데이터**는 내 고객사(특정 브랜드)를 위한 '쪽집게 과외'용으로만 쓴다.

이렇게 하면 데이터 확보 비용과 시간을 획기적으로(약 70% 이상) 줄일 수 있습니다.
