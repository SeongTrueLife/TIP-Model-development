T-Guardian 프로젝트 제안

AI 기반 상표 침해 자동 모니터링 시스템 (MVP Sprint)

Target: 2-Week MVP
Based on Doc v3.0 & Workflow v2.0

Pain Point

  • 등록 후 관리의 공백: 특허청은 등록만 해줌. 침해 감시는 전적으로 고객 책임.
  • 오프라인 사각지대: 지도, 간판, 로고 침해는 텍스트 검색으로 적발 불가능.
  • 인력의 한계: 변리사가 매일 네이버 지도를 검색할 수 없음.

Our Solution

"크롤러가 긁어오고, Vision AI가 판단한다."

  • 24/7 자동 감시 (네이버 지도/쿠팡)
  • 시각적 유사도(Visual Similarity) 분석
  • 변리사 의사결정 지원 (DSS)

System Architecture

1. Data Collection
Selenium Crawler (Naver Map)
2. Preprocessing
Nice Classification Mapping (업종 필터링)
3. Vision AI (Core)
YOLOv8 (Detection) + Siamese (Similarity)
4. Dashboard
React UI (변리사 판정 및 피드백)
