
# Node 0 (Calibrator) 튜닝 보고서 - 외관 모델 (Visual)

**작성일:** 2026-02-18
**대상 모델:** Model 2 (Visual)
**목표:** MVP 단계에서 오탐(False Positive)을 최소화하면서도, 데이터 기반 확률(48%)을 반영한 합리적 튜닝

## 1. 현상 분석 (미세한 겹침 구간)
*   **유사(Similar):** Raw Score 0.2 ~ 0.8 구간에 넓게 분포 (평균 0.49)
*   **비유사(Dissimilar):** Raw Score 0.0 ~ 0.6 구간에 넓게 분포 (평균 0.39)
*   **데이터 분석 결과:** Raw Score 0.4 ~ 0.7 구간의 실제 유사 확률은 약 **48%** 입니다.

## 2. 튜닝 전략 비교

### Option A: 정밀도 중심 (Hard Threshold) - *기각*
*   0.6점 -> 0.35점 (유사 아님)
*   너무 보수적이라 실제 침해 사례를 놓칠 위험이 큼.

### Option B: 재현율 중심 (Soft Threshold) - *기각*
*   0.6점 -> 0.55점 (가능성 열어둠)
*   실제 확률(48%)보다 높게 잡아 오탐 가능성 존재.

### Option C: 데이터 기반 타협안 (Hybrid) - *[최종 선택]*
*   **전략:** 데이터가 보여주는 실제 확률(48%)에 근접하되, 안전하게 살짝 낮춘 **0.45점**을 부여합니다.
*   **이유:** 애매한 구간에서는 스스로 판단을 유보하고(0.5 미만), **다른 모델(호칭/관념)의 점수가 높을 때만 침해로 넘어가는** 구조를 만듭니다.

## 3. 최종 앵커 포인트 설정 (Option C)

| Raw Score (입력) | Target Score (출력) | 설명 (의도) |
| :--- | :--- | :--- |
| **0.0** | 0.0 | - |
| **0.2** | 0.1 | 노이즈 제거 |
| **0.6** | **0.45** | **[핵심]** 회색지대(Gray Zone). 실제 유사 확률 48% 반영. 0.5 미만으로 억제하여 단독으로는 비유사 처리. |
| **0.7** | **0.65** | 가능성 구간 진입. |
| **0.8** | **0.85** | **[확신]** 여기서부터는 진짜 유사일 확률이 높음. 고득점. |
| **1.0** | 1.0 | 동일 |

> **기대 효과:**
> 1.  외관 단독으로는 애매할 때 '비유사(0.45)' 쪽에 가깝게判定.
> 2.  하지만 호칭/관념이 강력하면(예: 0.9), 평균 점수가 0.6~0.7로 올라가서 **'침해'** 포착 가능.
> 3.  데이터(48%)에 근거한 가장 논리적인 설정.
